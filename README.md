# 易文 · 微信文章助手

> 公众号创作者的全能 AI 助手 —— AI 写作、一键排版、素材中心、合规检测、多平台分发，让创作从 3 小时缩短到 30 分钟。

易文是一款 Chrome内核 浏览器扩展，在微信公众号后台编辑器中嵌入侧边栏，提供 AI 驱动的写作辅助、排版美化、素材管理、数据分析、合规检查和多平台一键分发能力。

## ✨ 核心功能

### ✍️ AI 写作
- **一键生成文章**：输入选题，选择行业和目标字数，AI 流式生成完整文章并插入编辑器
- **标题工坊**：AI 生成 5 个候选标题，附带风格标签和推荐理由，点击即可应用到编辑器
- **选中文本工具栏**：选中编辑器中的文字，悬浮工具栏提供润色、改写、扩写、缩写、续写 5 种 AI 操作
- **快捷键** `Ctrl+Shift+F` 一键 AI 排版

### 🎨 一键排版
- **6 套风格预设**：简约 / 商务 / 文艺 / 活泼 / 科技 / 教育，一键应用全文排版
- **品牌色定制**：10 种预设配色 + 自定义取色器，贯穿全文组件
- **AI 智能排版**：AI 分析文章内容，生成完整样式化 HTML 布局
- **排版微调**：滑块精确调整字号、行高、段间距、缩进、字间距

### 📦 素材中心
- **组件库**：7 大类（标题、引用、分割线、引导关注、行动引导、信息卡片、SVG 交互），30+ 精美组件一键插入
- **图片搜索**：接入 Unsplash 免费图库，搜索并插入高质量图片
- **文章采集**：右键任意网页 →「易文 - 采集此页面」，提取内容到编辑器
- **二维码生成**：输入链接或文本，生成二维码插入文章

### 📤 多平台分发
- **微信草稿/群发**：对接公众号 API，创建草稿、预览、群发、定时群发
- **一键分发 6 平台**：知乎、今日头条、掘金、CSDN、简书、百家号
- **手机预览**：手机外框样式预览最终排版效果

### 📊 数据分析
- **标题评分**：AI 从吸引力、好奇心、情感、清晰度、点击率 5 个维度打分
- **文章诊断**：AI 全面分析标题、结构、可读性、互动性、原创性，给出优化建议和最佳发布时间
- **阅读量预测**：AI 预估阅读量区间及影响因素
- **选题灵感**：AI 按行业生成 8 个热点选题

### 🔍 合规检测
- **本地检测，无需 AI**：扫描全文检查 4 类风险内容
  - 广告法违禁词（"国家级"、"最佳"、"唯一"等）
  - 极限词（"顶级"、"极致"、"完美"等）
  - 敏感词（"赚钱"、"震惊"、"转发"等）
  - 平台敏感词（"加微信"、"扫码加"等）
- 0-100 分评分，高/中/低风险分级，标注具体位置和修改建议

## 🤖 支持的 AI 模型

| 提供商 | 推荐模型 | 需要 API Key |
|--------|---------|:----------:|
| **DeepSeek** | `deepseek-chat`, `deepseek-reasoner` | ✅ |
| **OpenAI** | `gpt-4o`, `gpt-4o-mini` | ✅ |
| **Claude** | `claude-sonnet-4-6` | ✅ |
| **Ollama（本地）** | `qwen2.5:7b`, `llama3.1:8b` | ❌ |
| **自定义** | 任意 OpenAI 兼容接口 | 取决于配置 |

> 在设置页面选择提供商、填入 API Key 和模型名称即可使用。Ollama 本地部署无需 Key。

## 🚀 安装与使用

### 安装

1. 下载最新的构建产物（`dist/` 目录）或从 [Releases](https://github.com/hyper-share/yiwen-mp/releases) 下载
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择 `dist` 目录

### 使用

1. 进入 [微信公众号后台](https://mp.weixin.qq.com) 编辑器
2. 编辑器右侧自动出现易文侧边栏（或按 `Ctrl+Shift+T` 切换）
3. 在设置页面（⚙️）配置 AI 模型和 API Key
4. 开始使用各项功能

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+F` | AI 一键排版 |
| `Ctrl+Shift+T` | 打开/关闭侧边栏 |


### 技术栈

- **Vue 3.5** + **TypeScript 5.7** + **Pinia 3.0**
- **Naive UI** 组件库
- **Tailwind CSS** 样式
- **Vite 6** + **@crxjs/vite-plugin**（Manifest V3）
- **Chrome Extension Manifest V3**
